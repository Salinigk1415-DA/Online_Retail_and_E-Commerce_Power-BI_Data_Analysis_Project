
# ONLINE RETAIL & E-COMMERCE 



## Project Overview
This project focuses on analyzing an Online Retail & E-Commerce dataset using Microsoft Excel, Power Query, and Power BI. The aim is to transform raw transactional data into meaningful business insights through data cleaning, modeling, visualization, and dashboard development. The project helps businesses understand customer behavior, sales trends, product performance, and operational efficiency. Interactive dashboards and KPI reports enable stakeholders to make informed, data-driven decisions for business growth and improved customer satisfaction.
## Project Objectives
* Analyze online retail transaction data to identify business trends.
* Measure sales performance across product categories and cities.
* Understand customer demographics and purchasing behavior.
* Evaluate customer satisfaction using review scores.
* Build interactive dashboards for data-driven decision-making.
Create automated and scalable reporting solutions using Power BI.
## Dataset Description
The dataset is a cleaned CSV file containing 1,000 online retail transaction records collected during 2024–2025. It includes 13 attributes related to customer information, products, payment methods, order details, cities, age, gender, and review scores. The dataset covers five major categories:
* Electronics
* Sports & Outdoors
* Books & Stationery
* Fashion
* Home & Living

The data is fully complete with no missing values or duplicate records, making it suitable for advanced analytics and dashboard creation.
## Data Preparation & Transformation
Data preprocessing was performed using Microsoft Excel and Power Query to ensure data quality and consistency. Key steps included:
* Checking for null values and duplicates.
* Converting incorrect data types.
* Formatting the Order Date column into Date format.
* Creating derived columns such as Revenue, Month, Quarter, Year, and Age Group.
* Validating Review Scores, Quantity, and Price ranges.
* Trimming unnecessary spaces from text columns.
The cleaned dataset was then modeled in Power BI using a Star Schema approach with fact and dimension tables.
## Tools and Technologies
* Microsoft Excel – Initial data inspection and validation.
* Power Query – Data cleaning and transformation.
* Microsoft Power BI Desktop – Data modeling and dashboard creation.
* DAX (Data Analysis Expressions) – Calculated measures and KPIs.
* Power BI Service – Report publishing and sharing.
* CSV Files – Primary data source format.
## Analysis and Insights
The project generated multiple analytical insights through interactive dashboards:
* Electronics generated the highest revenue.
* Sports & Outdoors recorded the highest order volume.
* Female customers contributed more revenue compared to male customers.
* Customers aged 36–50 formed the highest-value segment.
* Cash on Delivery was the most preferred payment method.
* Customer satisfaction remained high, with most ratings being 4 or 5 stars.

The dashboards included KPI cards, bar charts, line charts, donut charts, maps, scatter plots, and slicers for advanced filtering and analysis.
## Key Outcomes
* Built a fully interactive Power BI dashboard for retail analytics.
* Automated reporting and reduced manual analysis efforts.
* Improved visibility into customer behavior and sales trends.
* Enabled better strategic planning through data-driven insights.
* Developed scalable data models for future business expansion.
## Conclusion
This project successfully transformed raw e-commerce transaction data into actionable business intelligence using Power BI. Through data cleaning, modeling, visualization, and KPI analysis, the project provided meaningful insights into sales performance, customer demographics, payment preferences, and satisfaction levels. The interactive dashboards support effective decision-making and demonstrate practical skills in data analytics, business intelligence, and visualization techniques.
